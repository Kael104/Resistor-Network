# Resistor-Network
A Python tool that analyzes resistor networks, calculates equivalent resistances, and visualizes the circuits using NetworkX and Matplotlib.

---

## Features

- Create **resistors, nodes, and networks**  
- Solve **series and parallel connections** automatically
- **Visualize networks**:  
  - **Black nodes** represent ground  
  - **Brown nodes** represent normal nodes
- Simple, readable Python API for building and analyzing networks  

---

## Input Requirements

- The network must have **at least 2 ground nodes**  
- The network must have **at least 1 non-ground node**
